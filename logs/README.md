# Logs Directory

This directory contains application logs generated by the 3Speak Upload Service.

## Log Files

- **error.log** - Error-level logs only
- **combined.log** - All log levels combined  
- **access.log** - HTTP request logs (if configured)

## Log Rotation

Logs are automatically rotated when using PM2:

```bash
pm2 install pm2-logrotate
pm2 set pm2-logrotate:max_size 10M
pm2 set pm2-logrotate:retain 30
```

## Viewing Logs

```bash
# View all logs
tail -f logs/combined.log

# View error logs only
tail -f logs/error.log

# View with PM2
pm2 logs 3speak-upload
```
